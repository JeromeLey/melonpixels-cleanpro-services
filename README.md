# CleanPro Services Demo Site

Professional cleaning services landing page for CleanPro Services, a demo site built by MelonPixels.

**Live Demo:** https://jeromeley.github.io/melonpixels-cleanpro-services/

## Tech Stack

- Next.js 15 + TypeScript
- Tailwind CSS v4
- Static export for GitHub Pages

## Pages

- **Home (/)** - Hero, services overview, trust indicators, testimonials, FAQ
- **Services (/services)** - All 7 cleaning service offerings
- **Packages (/packages)** - Home care plans and office contracts
- **Book (/book)** - Step-by-step booking form with price estimator
- **About (/about)** - Company background, staff vetting, contact info

## Development

```bash
npm install
npm run dev
npm run build
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on push to main branch.
